# Changelog

Notable changes to this project. Newest first.

## [1.0.0] - 2026-06-28
- First public release under the **AgentProof** name (formerly governance-in-a-box).
- Four self-verifying governance modules: autonomy gate (19/19), build discipline (22/22), model fitness (all-passed), agent-diff code review (23/23).
- `verify.py` runs every module's selftest as a subprocess and prints the real pass counts — stdlib-only, offline, no install.
